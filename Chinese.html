简介:

  • 刘羽，合肥工业大学仪器科学与光电工程学院教授、斛兵学者、博士生导师、生物医学工程系主任，国家级青年人才计划入选者，科睿唯安全球高被引科学家，IEEE Senior Member。2007-2016年就读于中国科学技术大学信息科学技术学院，获工学学士学位（2011）、工学博士学位（2016），曾获第30届中科大郭沫若奖学金、中国科学院院长优秀奖。2016年7月进入合肥工业大学工作，历任讲师、副教授、教授。目前主要从事图像处理、计算机视觉、机器学习相关方向的研究，具体包括图像融合、图像复原、医学图像分析等。作为项目负责人承担了国家自然科学基金重点项目合作单位课题/面上项目（2项）/青年基金、国家重点研发计划子课题、安徽省自然科学基金、商汤青年科研基金等多项科研项目。近年来，以第一/通讯作者在IEEE TPAMI / TIP / TNNLS / TCSVT / TGRS / TIM / TCI / TBME / JSTSP / JBHI / SPL、IJCV、MedIA、INFFUS、PR、IEEE/CAA JAS（自动化学报（英文版））、中国图象图形学报等国内外知名学术期刊和会议上发表论文70余篇，总计发表论文100余篇。论文谷歌学术总计被引20000余次，3篇第一作者论文单篇被引超1000次，16篇第一/通讯作者论文入选ESI高被引论文（5篇入选ESI热点论文）。个人入选科睿唯安全球高被引科学家（2023、2024、2025）、爱思唯尔中国高被引学者（2020至今）、全球前2%顶尖科学家（2020至今）。获中国图象图形学学会（CSIG）青年科学家奖（2025）、吴文俊人工智能优秀青年奖（2023）、安徽省自然科学二等奖（2019，3/3）、中国电子学会自然科学二等奖（2023，3/5）、中国仪器仪表学会金国藩青年学子奖（2022）、国际期刊IEEE Transactions on Instrumentation and Measurement年度唯一最佳论文奖（2020，通讯作者）、国际期刊IET Image Processing年度唯一最佳论文奖（2017，第一作者）。获评安徽省优秀青年研究生导师，指导的研究生中7人获评研究生国家奖学金，8人获评合肥工业大学优秀毕业生（其中3人同时获评安徽省优秀毕业生）。担任IEEE Transactions on Image Processing、Information Fusion、IEEE Signal Processing Letters、中国图象图形学报等多个国际国内知名期刊编委。担任中国人工智能学会（CAAI）智能融合专委会常务委员、中国图象图形学学会（CSIG）机器视觉专委会委员、中国图象图形学学会（CSIG）多媒体专委会委员、中国图象图形学学会（CSIG）青年工作委员会委员、中国图象图形学学会（CSIG）出版工作委员会委员、中国图象图形学学会（CSIG）合肥会员活动中心执行委员、医学图像计算青年研讨会（MICS）委员、安徽省人工智能学会多源信息融合专委会主任委员。

研究方向:

    图像处理、计算机视觉相关方向，主要包括多源图像融合与感知、图像复原、医学图像处理与分析、脑电信号处理等
  • 多源图像融合与感知：
    探索红外与可见光图像、医学图像、多聚焦图像以及多曝光图像的融合方法与感知技术。
  • 图像复原：
    研究雨、雪、雾等极端天气条件下，以及模糊、噪声、低照度、低分辨率等实际场景中的图像复原方法。
  • 医学图像处理与分析：
    围绕真实医学场景中的语义分割、目标检测与分类问题开展研究。
